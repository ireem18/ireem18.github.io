Merhaba size bu yazımda React nedir?Neden kullanmalıyız?Nasıl başlarız? gibi sorularınızda yardımcı olmaya çalışacağım.

React kullanıcı arayüzü oluşturmaya yarayan açık kaynak kodlu web uygulama çatısı. Facebook önderliğinde bir geliştirici grubu tarafından geliştirilmektedir.Opensource Javascript kütüphanesidir.

Hikayesine gelecek olursak:

Facebook şirketi Reactı geliştiriyor ve web sitesinde kullanmakta.Bu sırada Zuckerberg Instagram için bir web sayfası istiyor.Facebookun geliştirmekte olduğu bu kütüphaneyi instagram çalışanları "Facebook'tan bağımsız hale getirelim ve intagram web sayfasını aynı özellikleri kullanarak yapalım" diyorlar.İnstagram web sayfasının tamamı ReactJS ile yapılıyor.Daha sonra bu javascript kütüphanesini bütün herkese ücretsiz olarak paylaşalım diyorlar.

Interactive UI tasarımlarını acısız bir şekilde yapmanızı sağlar.

Reactın resmi sitesindeki tanımı budur. React sayfamızı her dafasında yeniden yüklemeden sadece değiştirilen verilerde değişiklik yapmamıza olanak sağlamaktadır. Web ve mobil uygulamalar için görüntüleme katmanı işlemek için kullanılır. React ayrıca yeniden kullanılabilir UI componentleri oluşturmamızı sağlar.HTML ve JavaScript’te temel bilgisi olan herkes React’ı kolayca anlayabilirler.React Native; IOS ve Android gibi mobil uygulamalara tepki mimarisi sağlayan 2015 yılında Facebook tarafından duyurulan native uygulamalar geliştirilmesini sağlayan bir kütüphanedir. React Virtual Dom denilen yapıyı kullanmaktadır.

Peki Bu Virtual Dom Nedir?

Performans arttırıcı bir yapıdır.JQuery veya Javascript geliştirilirken Real Dom üzerinde işlemler yapılıyordu.Yani değişiklik olmayan yerlerdetekrardan taranıp ve render edilmekteydi. Fakat React Virtual Dommimarisini kullanarak update özelliği ile sadece değişiklik olunan yer render edilmektedir.Böylece Real DOM'a a dokunarak maliyet azaltılır.React component tabanlı bir kütüphanedir.

React Kullanma Sebeplerimiz Nelerdir?

*Basitlik
*Öğrenmesi Kolay
*Native Yaklaşımı
*Performans
*Test Edilebilirlik

React Projesi Nasıl Oluşturulur?

Konsole ya da Ubuntu üzerine aşağıdaki komutu yazdığımızda belirttiğimiz adrese yeni bir react projesi oluşturacaktır.

Projemiz oluştuktan sonra Visual Stdio kullananlar için proje dizinine gidip code . komutuyla İDE de projeyi başlatmış olacağız.

VS code da projemiz açıldıktan sonra terminale yazacağımız npm start komutu ile React projemizi başlatmıs olacağız.

Keyifli kodlamalar dilerim :)
